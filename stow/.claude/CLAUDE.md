## Workflow Tips
- `cd` しないで。パスを毎回書いて。

## Coding Rules
- すべてのコードファイルは短く保って。10~50行が理想、最大100行まで。
- 古いファイルは、すべて削除して。このとき、新しいファイルに機能の移行が完了していることを確かめてから削除して。

## Domain Rules
### HTML
- 無駄な Element のネストは避けて、ミニマルな階層構造に保って。
